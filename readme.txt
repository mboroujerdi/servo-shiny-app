This repository contains the source code for a shiny app. The app simulates the classical servo mechanism together with its equivalent compartmental version. The compartmental version is used for the simulation of substrates and hormone kinetics. 

This application uses the shiny package in R. First, the reader should install R from www.r-project.org, then use the "Rstudio" IDE. The next step would be to download two files from the repository ui. R and server. R, it is advisable to open the IDE and create a new project. The next step would be to open the files ui. R and server. R, then install any libraries listed at the top of those files. The next step would be to press the run app in the IDE.
